================
Installing raking
================

Python version
--------------

The package :code:`raking` is written in Python
and requires Python 3.10 or later.

:code:`raking` package is distributed at
`PyPI <https://pypi.org/project/raking/>`_.
To install the package:

.. code::

   pip install raking

For developers, you can clone the repository and install the package in the
development mode.

.. code::

    git clone https://github.com/ihmeuw-msca/raking.git
    cd raking
    pip install -e ".[test,docs]"
